# Favicon Generation Instructions

To generate favicons for the portfolio:

## Option 1: Using Online Tool (Recommended)
1. Visit https://realfavicongenerator.net/
2. Upload a square logo/image (512x512px or larger)
3. Generate favicons for all platforms
4. Download and extract the zip file
5. Copy all files to `client/public/` folder

## Option 2: Manual Creation
Create the following files in `client/public/`:
- favicon.ico (16x16, 32x32, 48x48 multi-size)
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png (180x180)
- android-chrome-192x192.png
- android-chrome-512x512.png
